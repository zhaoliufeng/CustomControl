# CustomControl
zhaoliufeng.me/CustomControl

平时项目中使用到自订控件，包括：

1.LineChart 纵向增长柱状图 <br>
2.Histogram 横向增长柱状图<br>
3.DrawerPlusLayout 自定义侧拉控件<br>
4.BootPage 引导侧滑页<br>
5.NumTexView 数字增长控件<br>
6.DashBoardView 仪表盘控件<br>
7.PayPwView 支付密码输入框<br>
