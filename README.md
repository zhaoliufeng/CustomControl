# CustomControl
zhaoliufeng.me/CustomControl

平时项目中使用到自订控件，包括：

1.LineChart 纵向增长柱状图 
2.Histogram 横向增长柱状图
3.DrawerPlusLayout 自定义侧拉控件
4.BootPage 引导侧滑页
5.NumTexView 数字增长控件
6.DashBoardView 仪表盘控件
